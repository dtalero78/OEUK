# OEUK Medical History Application

Aplicación de historia clínica para OEUK con formulario de pacientes y panel médico.

## Estructura del Proyecto

- `backend/` - API REST con Node.js, Express y SQLite
- `frontend/` - Aplicación React con Vite

## Instalación

### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

## Características

- Formulario de screening médico para pacientes
- Panel administrativo para médicos
- Base de datos SQLite para almacenamiento
- Autenticación simple para panel médico
